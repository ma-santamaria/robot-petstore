# Robot framework example

Simple example to know the bases of [Robot][1].

The [RequirestsLibry][2] library is used.

The same examples implemented are the same that can be found in [this example with Karate][3].

1. User creation and recovery of its data.
2. Recovery of the names of the sold pets and their frequency.

## Execution

The tests can be executed with `Robot Tests`

## Report

You can look at the report in HTML format on the file `report.html`
